🎯 Clasifica cada sistema
Para cada sistema, decide si tiene solución única, infinitas soluciones o si es inconsistente (sin solución). La retroalimentación aplica el Teorema del Rango del libro: si R(A) = R(A|B) = n hay solución única; si R(A) = R(A|B) < n hay infinitas soluciones con n − R(A) parámetros; si R(A) < R(A|B) el sistema es inconsistente y S = ∅. Incluye dos ejemplos resueltos del libro. 10 preguntas por ronda, dificultad progresiva.
