📝 Matriz A
Ingresa una matriz cuadrada A. La herramienta calcula el polinomio característico P(λ) = det(A − λI), encuentra los eigenvalores, y para cada uno resuelve el sistema (A − λI)v = 0 para obtener los eigenvectores y el subespacio propio Vλ.
